Deploy MPICH3 on Ubuntu Linux (Multi-Node Cluster)

MPICH (all machines)

Download MPICH from mpich.org and extract the contents of the MPICH package to a location of your choice, then compile the source code to binaries. .

tar xzvf mpich-3.1.tar.gz

cd mpich-3.1.tar.gz

Choose an installation directory for MPICH, I pick /usr/local/mpich. Next, we compile and install MPICH.

./configure -prefix=/usr/local/mpich

make

make install

Add the following lines to the end of the $HOME/.bashrc file.

export MPICH_HOME=/usr/local/mpich

export PATH=$PATH:$MPICH_HOME/bin
